main_menu(user, send_msg)¶

main menu for response to Reception PI. option 1 is search function, using socket communicate with Flask API get the search result and then return it to the Reception PI. option 2 is borrow function, get the json send by Reception PI, and send the book detail to the Flask API, then get the result and send it back to the Reception PI. option 3 is normal return function, if Reception PI send a return book request, master pi should find the corresponding borrowed information from google cloud sql through Flask API, the remove this record. option 4 is barcode return function, get the barcode information from Reception PI, and find out the corresponding borrowed information and remove it. option 5 is log out function, the master pi will send back the close message to reception pi to close this socket

main_menu(usr_name)¶

option 1, send a search request to master pi, type e to exit. option 2, send a borrow request to master pi, the format should follow the instruction. option 3, show the book this user has already borrowed, send a normal return request to master pi, used for returning a book. option 4, scan a barcode instead of typing the book detail to return a book option 5, log out the system
